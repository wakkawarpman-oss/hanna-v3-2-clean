# Installation Instructions for Metagoofil

Metagoofil is a metadata harvester tool for extracting metadata from public documents available on target websites. Follow these steps to install it manually:

## Prerequisites
- Python 3.x
- Git

## Steps
1. Clone the repository:
   ```bash
   git clone https://github.com/laramies/metagoofil.git
   ```

2. Navigate to the directory:
   ```bash
   cd metagoofil
   ```

3. Install dependencies (if any):
   ```bash
   pip install -r requirements.txt
   ```

4. Run the tool:
   ```bash
   python metagoofil.py -h
   ```

## Notes
- Ensure Python is added to your PATH.
- If you encounter issues, check the repository README for troubleshooting tips.
